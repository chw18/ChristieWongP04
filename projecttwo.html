User-oriented Application

Navigating within Granville Island

Role
Graphic Designer, Animator & Programmer

Client
Simon Fraser University

Year / Time Taken
2011 / 6 Weeks

Tools
Adobe Illustrator, Flash, Photoshop

This was the final project for an Information Design course. The purpose of this project was to design a way-finding tablet application for a specific user to navigate within a space, using the concepts of mapping strategies, icon representation, and user-orientation.

This is a walkthrough video of the way-finding application my team created. We chose to tailor this application for a six-year old child to navigate Granville Island, using vibrant colours, simple icons, and minimal text. I drew a portion of the graphics using Illustrator, coded the action-script and designed the animation in Flash. We also met up with the professor and teaching assistant for weekly feedback. As a result, we devised a Flash prototype of this application, successfully incorporating the concepts learned from this course.
